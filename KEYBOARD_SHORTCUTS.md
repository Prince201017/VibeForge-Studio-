# ForgeOS Keyboard Shortcuts Guide

## General

| Action | Shortcut |
|--------|----------|
| Open Command Palette | `Ctrl+K` / `Cmd+K` |
| Escape / Cancel | `Esc` |

## Edit

| Action | Shortcut |
|--------|----------|
| Undo | `Ctrl+Z` / `Cmd+Z` |
| Redo | `Ctrl+Shift+Z` / `Cmd+Shift+Z` / `Ctrl+Y` |
| Cut | `Ctrl+X` / `Cmd+X` |
| Copy | `Ctrl+C` / `Cmd+C` |
| Paste | `Ctrl+V` / `Cmd+V` |
| Delete | `Delete` / `Backspace` |
| Select All | `Ctrl+A` / `Cmd+A` |
| Deselect | `Escape` |

## Tips

- Press `Escape` to dismiss any dialog or cancel current operation
- Use Command Palette (`Ctrl/Cmd+K`) to search and run any command
- Keyboard shortcuts work from anywhere except when editing text
- Middle-click or spacebar to pan the viewport
- Scroll wheel to zoom in/out on the canvas
